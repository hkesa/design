Design Terminology

UI (User Interface)

The visual layout of the website. It includes buttons, images, text, and how the site looks to the eye.

UX (User Experience)

How the user feels when navigating your site. Is it easy to find the contact button? Does it flow logically?

CMS

Content Management System like WordPress or Shopify that allows you to edit content without coding.

Hosting

The "land" where your website house sits. We handle secure server setup.

Golden Rule

Design principles that ensure visual balance and focus, guiding the user's eye to the most important content.

SSL (HTTPS)

Secure Sockets Layer. It encrypts data, providing the green lock icon. Essential for security and Google ranking.

RWD

Responsive Web Design. Allows your site to adapt perfectly to mobile, tablet, and desktop screens automatically.

SEO

Search Engine Optimization. Techniques to help your website rank higher on Google search results.

UTM Tags

Codes added to URLs to track where your traffic comes from (e.g., from a specific Facebook ad).

Meta Tags

Snippets of text that describe a page's content; they don't appear on the page itself, but only in the page's code.

HTML/CSS/JS

The building blocks of the web. HTML is structure, CSS is style, and JavaScript is interactivity.

Google Search Console

A free service by Google that helps you monitor, maintain, and troubleshoot your site's presence in Google Search results.

Google Business Profile

Your business listing on Google Maps and Search. Crucial for local SEO.

Google Analytics

A web analytics service offered by Google that tracks and reports website traffic.

Facebook Pixel

Code placed on your website that helps you track conversions from Facebook ads.

Landing Page

A standalone web page, created specifically for a marketing or advertising campaign.

Sales Funnel

The journey that a potential customer goes through on the way to purchase.
